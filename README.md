🚗 Choudhary Travels - Car Booking System
A full-stack web application for online car booking built using Django.
This platform allows users to book rides easily and enables the owner to manage bookings through an admin dashboard.

🌐 Features
🧾 Online car booking system
⏱ Automatic time-based pricing
❌ Booking conflict detection (no double booking)
📲 WhatsApp integration for instant booking notification
🧑‍💼 Admin panel to manage cars and bookings
🖼 Upload car images
📱 Fully responsive UI (mobile + desktop)
🎨 Modern UI with animations (AOS)
🛠 Tech Stack
Frontend: HTML, CSS, Bootstrap, JavaScript
Backend: Django (Python)
Database: SQLite
Deployment: Render
Version Control: Git & GitHub
